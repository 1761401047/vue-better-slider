# vue-better-slider

## 这是什么(What is it)
为vue提供的一个轮播图组件

## 代码演示如何使用

npm install vue-better-slider --save 

1.导入并注册组件,注册组件
```js
import {Slider} from 'vue-better-slider'
components: {
   Slider
}
```
2.template模板
```js
  <slider>
      <div >
        <a href="">
          <img scr="">
        </a>
      </div>
  </slider> 
```
## 组件演示demo

```js
git clone https://github.com/songhaoreact/vue-better-slider.git
cd vue-better-slier
npm install 
npm run dev
```
## 说明

1. 改组件是根据better-scroll滑动组件封装改成

2. 实现功能有：轮播 自动播放 dots 循环播放，适合手机端,图片高度自适应
